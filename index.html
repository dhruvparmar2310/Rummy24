Card Game

Game Rules :

  • Jo suits/group id/joker same hoi sequence ma to teno score "0"
  • Jo suits/group id/joker same card na hoi to teni value add karavani
  • Jo suits/group id/joker same na hoi to A/J/Q/K ni value 10 levani respectively...
  • Jo A/J/K/Q line ma hoi to aa group id no score "0", (1/2/A/J/K/Q sequance ma nathi atle value add thase A=10)
  • Jo je card ma joker hoi, to te card ni value zero levani.
